🚀 Getting Started

1️⃣ Clone the Repository
git clone https://github.com/Mehtaab-010110/My-Portfolio.git
cd My-Portfolio

2️⃣ Install Dependencies
npm install

3️⃣ Run the Development Server
npm run dev

Open http://localhost:3000 in your browser to view the project.
